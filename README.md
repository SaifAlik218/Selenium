# Selenium Automation Project

This project contains Selenium WebDriver automation test scripts developed using Java.

## Technologies Used

- Java
- Selenium WebDriver
- ChromeDriver
- Test Automation
- Git & GitHub

## Project Features

- User Signup Automation
- Form Validation Testing
- Dropdown Handling
- Radio Button Handling
- Waits Implementation
  - Explicit Wait
  - Fluent Wait
- Screenshot Capture Utility
- Package-based Framework Structure

## Project Structure

```bash
src/
│
├── automationExercise/
├── blaze/
├── locator/
├── takeScreenShot/
```

## Concepts Practiced

- Inheritance
- WebElement handling
- Waits
- JavaScriptExecutor
- Select class
- Exception Handling
- XPath Locators

## How to Run

1. Clone the repository

```bash
git clone https://github.com/SaifAlik218/Selenium.git
```

2. Open project in Eclipse/IntelliJ

3. Add Selenium dependencies

4. Run the main class
